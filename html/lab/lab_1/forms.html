Форми:

Форма - елемент веб сторінки, призначений для отримання даних від користувача. Парний тег <form> являє собою контейнер, який може місти будь-які інші елементи, але у першу чергу призначений для розміщення елементів вводу:

  • - <input>
  • - <select>
  • - <textarea>
  • - <button>

Також, існують специфічні для форми елементи, які не призначені для введення даних, але за своїм призначенням тісно пов'язані із названими вище елементами:

  • - <label>  Визначає мітку для елемента <input>
  • - <fieldset>  Групує пов’язані елементи в формі
  • - <legend>  Визначає заголовок для елемента <fieldset>
  • - <optgroup>  Визначає группу пов’язаних параметрів в випадаючому списку
  • - <option>  Визначає параметр в випадаючому списку
  • - <datalist>  Визначає список попередньо визначених параметрів для елементів управління вводом
  • - <output>  Визначає результат розрахунку

Найважливішим та найчастіше вживаним елементом форми є елемент <input>. У залежності від атрибуту type цей елемент може відображатись багатьма способами:

  • - <input type="button">
  • - <input type="checkbox">
  • - <input type="color">
  • - <input type="date">
  • - <input type="datetime-local">
  • - <input type="email">
  • - <input type="file">
  • - <input type="hidden">
  • - <input type="image">
  • - <input type="month">
  • - <input type="number">
  • - <input type="password">
  • - <input type="radio">
  • - <input type="range">
  • - <input type="reset">
  • - <input type="search">
  • - <input type="submit">
  • - <input type="tel">
  • - <input type="text">
  • - <input type="time">
  • - <input type="url">
  • - <input type="week">
