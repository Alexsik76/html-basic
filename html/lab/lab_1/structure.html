Структура проекту:

Проект складається зі статичного index.html файлу, основного файлу стилів style.css та файлу скриптів мовою JavaScript main.js. На стадії початкового завантаження сторінки лівий блок кнопок <aside id="accide_buttons"class="aside-left nav nav--coll"> - порожній. Так само не завантажується вміст секції <section id="inserted" class="main-content"> Наприкінці завантаження статичного контенту, завантажується js скрипт, який починає виконуватись:

  • - завантажуються данні про структуру проекту з файлу project_structure.json. Цей файл містить відомості про кнопки лівої панелі для кожної лабораторної роботи з їх написами та назвою файлів html та css, що відповідають кожній кнопці;
  • - кнопкам вибору лаборатрної роботи присвоюється обробник події натискання: 
    for (button of all_buttons) {
        if (button.id.startsWith("lb")) {
          button.addEventListener("click", lbButtonHandler);
        }
      }
    function lbButtonHandler(event) {
    current_lb = site_data.labs[event.target.id];
    setCurrentBtnActive(new_btn = event.target);
    for (button in current_lb.buttons) {
        createAcideButton(button, current_lb.buttons[button]);
    }
        ...
    }
  • - динамаічно генерується html-код кожної кнопки;
  • - відбувається завантаження вмісту основного блоку;

У подальшому натискання кнопок лівого блоку завантажують до основного блоку вміст відповідних html файлів а також замінюють посилання на відповідний css файл.
