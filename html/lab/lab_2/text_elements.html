Стильове оформлення текстових елементів, зокрема списків.

Списки бувають нумеровані <ol>та не нумеровані <ul>

У обох випадках список має містити хоча б один елемент <li>. При потребі розмітисти вкладений список, він має бути всередені відповідного тегу <li>, а не між ними.

Для списків існують спеціальні властивості:

  • list-style-image - картинка перед кожним елементом списку;
  • list-style-position - розташування маркерів в межах границь списку або поза ними;
  • list-style-type - тип маркера (наприклад, для нумерованих списків тип lower-roman встновить нумерацію римськими цифрами).

Ці властивості можуть бути об'єднані скороченим записом: list-style: lower-roman url("img/shape.png") outside;

Код для списку више:

      li {
        list-style-type: "* ----------------> ";
        list-style-position: inside;
    }

Нумерований список для прикладу:

  1. Сципіон
  2. Гай Юлій Цезар
  3. Тіберій
  4. Нерон

Використано код:

      ol li {
        list-style-type: lower-roman;
    }
