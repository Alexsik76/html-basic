CSS селектори використовуються для "пошуку" (або вибору) HTML-елементів, які необхідно стилізувати.

Селектор тегу призначений для вибору елементу за назвою тегу.

Тобто, наприклад код 

                p {
                    text-align: center;
                    color: red;
                  }
буде застововано до усіх тегів <p> на сторінці.

Під час виконання даної роботи такі селектори застосовувались, наприклад, для стилізації заголовків та посилань: 

            a {
                border-bottom: 1px dotted;
            }
            h1,
            h2,
            h3 {
                font-weight: 700;
            }

 Перед тим, як переходити до наступного виду селекторів, слід відзначити наступне: Один елемент може підпадати під декілька правил, описаних в таблиці стилів. І ці правила можуть суперечити одне одному. Тому браузери мають алгоритм, за яким обирають лише одне з конкуруючих правил. У загальному випадку застосвується останнє значення властивості описане в таблиці стилів. Однак, важливо знати, що інує ще і алгоритм спеціфічності, який обчислює вагу селектора. Цей алгоритм працює наступним чином: кожен селектор отримує табличку з трьома значеннями "ID - CLASS - TYPE". При цьому вага ID більша ніж CLASS, у свою чергу CLASS має більшу вагу, ніж TYPE.

Наприклад, маємо елемент:

            <div class="usualy-div" id="only-one"> Якийсь текст </div>

При застосуванні стилів:

           div {
            color: red;
           }
           div {
            color: green;
           }
           div {
            color: blue;
           }

текст буде синім, тому що при рівнозначих селекторах застосується останній.

Однак, при такому виборі:

            #only-one {
              background-color: red;
            }
            .usualy-div {
              background-color: green;
            }
            div {
              background-color: blue;
            }

спрацює перший селектор, як найбільш вагомий (1-0-0).
