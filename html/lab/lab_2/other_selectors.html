Сусідні, дочірні, універсальні та селектори атрибута.

На практиці часто виникає потреба у виборі певної підмножини елементів, однак не завжди доцільно додавати зайві класи.

У таких випадках може стати у нагоді один із перелічених у заголовку видів селекторів.

Так, універсальний селектор * застосовується для вибору усіх елементів на сторінці.

У даному проекті використовувася для обнулення відступів і границь:

      * {
        padding: 0;
        margin: 0;
        border: 0;
      }

Для управління стилем сусідніх елементів використовується символ плюса (+), який встановлюється між двома селекторами E і F:
E + F {Опис правил стилю}. Стиль при такому записі застосовується до елементу F, але тільки в тому випадку, якщо він є сусіднім для елемента E і розташований відразу після нього.

Для звуження області дії селекторів доцільно застосовувати дочірні селектори. Синтаксис такий, що спочатку записується батьківський силектор а після нього через пробіл - дочірній. 

        .inserted__content p {
              padding: 0.5em;
        }
У даному випадку вказаний внутрішній відступ буде застосвано до тегів абзацу, які розташовані всередині елементу з класом ".inserted__content". Наведений код ідентичний наступному: 
        .inserted__content {
          p {
              padding: 0.5em;
          }
        }

Іноді доводиться використовувати селектори атрибута. Їх синтаксис включає доволі багато гнучких правил. Вибір елемент відбувається за значенням атрибута.

Наведу деякі приклади для пояснення:

  • [attr] - елемент з атрибутом "attr";
  • [attr=value] - значення атрибута дорівнює value;
  • [attr|=] - дорівнює або починається з value;
  • [attr^=value] - починається з value;
  • [attr$=value] - закінчується на value;
  • [attr*=value] - містить хоча б одне value.

Наприклад, так обирались усі кнопки вибору лабораторних робіт:

        for (button of document.querySelectorAll("button[id^='lb']")) {
          button.addEventListener("click", lbButtonHandler);
        }
